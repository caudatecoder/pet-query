## Setup

* install ruby 3.4 on your system
* `bundle install`
* `rails db:create`
* `rails db:migrate`
* `rails s`

To run tests:
 * Make sure the test database is up
   * `RAILS_ENV=test rails db:create`
   * `RAILS_ENV=test rails db:migrate`
 * `rspec` or `rspec _filepath_`
